# 🌐 Personal Portfolio Website

## 📌 Project Overview

This project is a modern and responsive Personal Portfolio Website that showcases my professional background, skills, education, qualifications, certifications, work experience, and projects. It was created to present my experience in a professional way and can be shared with potential employers during job applications.

The website is easy to navigate, works on desktop, tablet, and mobile devices, and demonstrates my ability to build and organize a professional web project.

---

## ✨ Features

* 🏠 Home page with a professional introduction
* 👤 About Me section
* 💻 Technical and Soft Skills
* 📂 Projects section with three projects
* 🎓 Education
* 📜 Certifications
* 💼 Work Experience
* 📞 Contact information
* 📱 Responsive design for all devices
